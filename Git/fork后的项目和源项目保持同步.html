fork后的项目和源项目保持同步

  1. 保证fork后的仓库和本地仓库同步
  2. git remote -v 查看所有远程仓库
  3. 若没有源仓库 git remote add upstream URL
  4. git fetch upstream 从远程源仓库拉取
  5. git checkout master 切换到master分支
  6. git merge upstream/master 合并远程源仓库和本地仓库
  7. git pull upstream master 拉取远程仓库更新
  8. git push -u origin master 把仓库更新推到fork后的仓库上
